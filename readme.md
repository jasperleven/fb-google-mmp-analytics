# **Marketing Analytics Project**

## **Описание проекта**

Проект включает сбор и синтез данных по рекламным кампаниям, построение модели данных, расчет ключевых метрик и визуализацию в Power BI.  
 Все этапы выполнены и распределены по отдельным папкам с инструкциями.

## **Выполненные задачи**

### **Сбор данных**

* Синтетические данные за последние 14 дней по Facebook Ads и Google Ads: `date`, `campaign_id`, `campaign_name`, `impressions`, `clicks`, `spend`, `installs`.

* Агрегаты MMP (Adjust / AppsFlyer) по кампаниям: `installs`, `d1_revenue`, `d7_revenue`.

* Файлы находятся в `/сбор данных`.

* Для генерации данных используется скрипт `/сбор данных/synthetic_ads_script.py`.

### **Модель данных**

* Канальные данные сведены с MMP по `campaign_id`.

* Рассчитаны метрики:

  * CTR \= clicks / impressions

  * CPI \= spend / installs

  * ROAS D1 \= d1\_revenue / spend

  * ROAS D7 \= d7\_revenue / spend

* Логика расчёта и SQL-скрипт находятся в `/модель данных`.

### **Дашборд в Power BI**

* Визуализации:

  * Карточки KPI: Spend, Installs, Clicks, Impressions, CTR, CPI, ROAS D1 / D7

  * Диаграмма по дням: Spend, Installs, Clicks, Impressions

  * Таблица по кампаниям: Spend, Installs, Clicks, Impressions, CPI, ROAS D1 / D7

  * Срезы (Slicers): Источник (FB/Google), Период

* Дашборд сохранён как `/Дашборд в Power BI/Power BI.pbix`.

### **Итоговые выводы**

* Файл `/Итог/Выводы.md` содержит рекомендации по масштабированию и паузе/перезапуску кампаний на основе рассчитанных метрик.

## **Как использовать проект**

1. **Генерация данных**

   * При необходимости запустить `/сбор данных/synthetic_ads_script.py` для создания CSV-файлов.

2. **Создание модели и расчёт метрик**

   * SQL-скрипт `/модель данных/final_script.sql` объединяет каналы, связывает с MMP и считает ключевые показатели.

3. **Визуализация в Power BI**

   * Открыть `/Дашборд в Power BI/Power BI.pbix`

   * Использовать срезы для анализа по источникам и периоду.

4. **Выводы**

   * Ознакомиться с `/Итог/Выводы.md` для краткой аналитики кампаний.
